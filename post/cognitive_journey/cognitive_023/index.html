【真实世界认知补充（三）】
《房间里的成年人》总结（上）：有关第一轮、第二轮纾困贷款

凭借极慢的阅读速度（大概3min/页）我终于读完了《房间里的成年人》。这个过程是比较痛苦的，一方面是因为还没培养起长篇阅读所需的平心静气，总得把自己按在板凳上、除掉心里生长的野草；另一方面更加是因为瓦鲁法克斯与“三驾马车”的博弈困境实在如同一个黑洞，会吸纳走一切好心情：
当你做了充足的准备工作、精心起草了方案、准备打一场翻身仗时，最终却发现你的对手捂住了他自己的耳朵，既听不进去你的观点也提不出任何自己的办法，坐看这僵局一拖再拖。与此同时，你最亲爱战友的意志“再而衰、三而竭”，不仅背叛了你、甚至开始准备投降了！这是一种怎样的无能为力与折磨感啊。

从这一篇认知之旅开始，让我们重新回到那些时刻，看一看究竟发生了什么。

*注：这里的“三驾马车”指欧盟委员会、欧洲中央银行、国家货币基金组织。

金融危机之前

2008年金融危机爆发之前，希腊由于长期发展落后、管理不善、腐败，已经拥有了欧盟中相对收入来讲最深的债务之洞。但“在现实中，政府永远不会偿还债务。他们将贷款展期，也就是无休止地推迟还款，只支付贷款利息。只要能继续这样做，他们就仍有偿付能力”。

直到2008年初，希腊的收入还在以5.8%的速度迅速增高，而债务之洞加深的速度仅为4.4%，造成了一种可持续的表象。而第二年这两个数字便颠倒过来。

第一轮纾困贷款

认知 020摘录了原书起始篇章的部分内容，讲述了希腊债务危机起因、欧盟欧元本身制度存在的缺陷，并向我们展示了真实世界的荒谬：所谓“第一轮纾困”实则是为拯救法德银行家而包装的谎言。这里再次回顾一下：

噩梦的连锁路径可能是这样的：金融危机导致欧洲银行家入不敷出、叫停贷款 -> 导致希腊政府和企业借不到钱、无法继续推动收入增长而面临破产 -> 进一步导致当初放贷无度、风险敞口巨大的三家法国银行，因希腊等“欧猪五国”的坏账而变得资不抵债、急需法国政府救援 -> 可是这些债务的规模至少是法国经济总量的几倍，只会导致法国政府自己的债务变成黑洞。与此同时，德国也在相似的处境里挣扎：好不容易填上了金融危机造成的黑洞，现在也开始为希腊即将引发的问题头疼。

*注：欧猪五国（英语：PIIGS）是国际债券分析家、学者和国际经济界媒体对欧洲五个主权债券信用评级较低的经济体的贬称。这个称呼涵盖葡萄牙、意大利、爱尔兰、希腊、西班牙，特别指各国的主权债券市场，这些国家的公共赤字也都超过了3%。

上升周期留下的泡沫，在此时展露了可怕的真面目。

然而制度的缺陷在于：欧元集团形成后，各国不再拥有自己的央行，只能依靠欧洲共同的央行——欧洲中央银行，由此各国不再能实施自己的货币政策来处理债务危机（比如像美联储一样印钱，使债务货币化）。同时呢，联盟创建初期又有一条明文规定：希腊-拉丁地区的坏账，不论是私人的还是公共的，不得转移到欧洲央行的账面上。也就是说，央行可不替大家背锅哈！

不能自救、联盟也不管，希腊的债务危机最终就会扩散至法国、德国进而拖垮整个欧盟。所以......为了欧洲的未来，当然也是为了政府官员们的政治生涯不被此次危机断送......希腊不能承认破产！（也就不能正常进行债务重组/剃头）不仅如此，还要以它为借口拿到一大笔钱去救助前一秒还在挥霍无度的银行家们。从谁那里借到钱呢？欧洲最普通的老百姓们！又要谁来背负债务还钱呢？希腊最普通的老百姓们！建制派这套如意算盘真是打的啪啪响呢

*注：
- 债务重组（Debt Restructuring）：债务重组是指债务人与债权人之间达成协议，重新安排债务的支付条件和结构，以减轻债务人的负担。重组可能包括延迟还款期限、减少利息支付、转换债务形式等措施，以使债务更可管理和可支付。债务重组通常是基于协商和合作的基础上进行的，旨在避免债务违约和可能的法律程序。
- 债务剃头（Debt Haircut）：债务剃头是指债权人同意减少债务的本金或利息，以减轻债务人的压力。债务剃头通常是在债务违约或无法偿还的情况下发生，是一种紧急措施，通常与破产情况相关。

但这套逻辑本质可不能明说，怎么办呢？“聪明人”给它起了一个表面光鲜的新名字：纾困协议，即我们要大发善心、帮助挥霍懒惰的希腊同胞走出困境。

不仅名字要光鲜亮丽，实施过程也得看起来合理合法才行：
- 有规定说欧盟不得为各国政府债务提供融资，那就让国际货币基金组织加入，提供“国际贷款”而不是“欧洲贷款”；
- 又有规定说国际货币基金组织永远不向破产政府提供贷款，除非债务"剃头"——得到重组，正好当时的总裁是法国人，那就让他说服内部官员开个绿灯。
最终，法德银行家们弄出的窟窿变成了全世界的纳税人为其买单。

2010年5月初，第一轮纾困贷款签字生效，约1100亿欧元（其中1/3来自国际货币基金组织）在希腊上空转了一圈后立即被抽回法国德国银行。

让我们把视野转回希腊国内：

2009.09-2011.11这期间，乔治·帕潘德里欧领导的泛希腊社会主义运动（社会民主党）任希腊执政党，努力推动议会促成了第一轮纾困协议的签订，并大刀阔斧地实施紧缩措施。他们宣称这是希腊的救命稻草，而债务重组既不必要也不可取。

也许是因为他们惧怕被踢出欧元区、希望继续跟各国政府银行及希腊寡头保持和谐的关系，又或许他们不明白也不愿意相信签订的真实代价。

总之，由于这些协议与措施造成的灾难，该党的支持率在随后几年暴跌，越来越多的民众占领宪法广场抗议示威，最终出现“5000名警察包围宪法广场，使用数量前所未有的催泪瓦斯、眩晕弹、烟幕弹、高压水枪来驱散人群”的政府暴行。

可以说帕潘德里欧政府将自身的道德资本和政治资本消耗殆尽，注定垮台。

第二轮纾困贷款

法国和德国的银行得救了，到2012年底，希腊政府债券已完全在其账面上消失。而希腊却要继续为这笔巨额纾困贷款还账，可是它还得起吗？

没有地方储存坏账（央行不管），没有地方“开源”（印钱），就只能“节流”，这就是为什么欧洲的掌权者那么喜欢采取紧缩措施的原因。然而在大萧条的环境下，实行紧缩（削减工资和福利、取消工会权利、增加税赋）是致命的。

记住，我们不能用个人财务的视角来理解公共财政。瓦鲁法克斯写道：“在你的生活中，你的支出和收入是完全分开的。因此，当你削减开支时，你的收入不会减少。但如果整个国家大量储蓄，其总收入反而会下降。造成这种情况的原因在于，在国家层面，总支出和总收入是完全相等的，因为你赚到的都是别人花掉的钱。因此，如果一个国家的所有人和所有企业都在削减开支，政府一定不能也跟着削减开支。如果政府削减开支，总支出将陡然下降，国民收入也将随之陡然下降，接着导致财政部的税收收入下降，闹出紧缩措施的大乌龙：国民收入不断缩水，造成现有国债无法偿还。这就是为什么说紧缩措施绝对是错误的解决方案。”

原本法德银行持有的希腊债务，由于纾困贷款变成了各国政府持有的希腊债务，简单的商业问题就变成了政治问题。如果进行债务重组，政客们就需要回到议会承认自己之前撒过的谎、承认让本国政府和百姓背上了巨额坏账，但这无异于政治自杀。同时他们其实也不指望希腊政府能够还得起钱，因为“向破产实体提供更多贷款绝不可能使其恢复偿债能力，尤其是当贷款协议要求削减其收入的时候”。“唯一的选择是继续假装下去，再给希腊政府一笔钱，让希腊假装偿还拖欠欧盟和国际货币基金组织的债务：这就叫做第二轮纾困”。

然而，瓦鲁法克斯写到：“要是不对希腊债务进行某种形式的剃头， 将其作为2012年第二轮纾困的一部分，那么，将向希腊提供的贷款的惊人数额会让欧洲各国已然非常愤怒的议员们集体心梗发作。萨科齐总统和默克尔总理只得接受对希腊债务进行重组的主意，但条件是，损失只能由那些无力对法、德造成太大伤害的债主承担。2011年夏，方案敲定：债务剃头的损失主要由希腊养老基金、希腊半公共机构和购买了政府债券的希腊储户承担，国际货币基金组织和欧洲各机构在2010年提供的贷款自然仍是不可侵犯的。”

如果所谓的“债务重组”，真的是为了帮助希腊恢复偿付能力的话，那就还算有一丝希望。可惜事实并非如此。根据第二轮纾困方案的条款，新政府将宣布得到世界历史上最大的一笔不会拨付的贷款。“举世无双的1000亿欧元债务违约"剃头"重创了希腊无权无势的养老金领取者、专业协会和小额债券持有人（他们将被迫与政府欠他们的钱说拜拜），而创世界纪录的1300亿欧元"假意展期"贷款被强行记在这个国家名下，尽管希腊政府本身几乎得不到一分钱。其中很大一笔钱反而流入了希腊银行家的口袋（以过度补偿他们因政府债券"剃头"而承受的损失），还有一大笔钱给了向希腊提供贷款的外国私营贷款机构（以奖励他们接受债务"剃头"），第三笔钱则用于偿还欧盟和国际货币基金组织在第一轮纾困协议中提供的贷款。”

“由于把钱借给希腊政府，希腊养老基金、专业组织和小额债券持有人损失了多达90％的本金。与他们不同的是，希腊政府债券的外国持有人几乎没有损失。原因在于，早在2012年"剃头"开始之前，外国银行和其他机构投资者已经以折扣价将手中的希腊政府债券出售给欧洲央行或偏好风险的对冲基金。等到2012年，外国机构买入希腊政府债券的价格不足债券面值的30%。因此，把他们得到的接受"剃头"条款的"奖励"算在内，他们甚至从2012年的官方"剃头"中赚了钱。”

如果说第一轮纾困实际是为了拯救法德银行家，那第二轮纾困就是为了拯救希腊自己的银行寡头和外国投机者。唯独没有一项计划是为了拯救希腊自己的人民。

趁着帕潘德里欧彻底倒台之前，“三驾马车”还驱使他做了最后一次献祭：在2011年10月，签订了与其一贯主张恰恰相反的——以债务重组为基础的第二轮纾困贷款。

回到希腊国内来看，虽然帕潘德里欧倒台前在第二轮草案上签了字，但这项纾困方案还需要希腊议会的通过。“三驾马车”需要一个新的、听话的继任政府来完成此事，并监督和推动方案的落实。

此时希腊国内支持率最高的是安东尼斯·萨马拉斯领导的新民主党，之前他一直采取反紧缩、反纾困的立场，曾私下给瓦鲁法克斯打电话称赞并感谢他所持的见解和立场。

然而不知道默克尔（时任德国总理）施展了什么神奇的魔法，萨马拉斯在仅仅与她见了一次面之后便放弃了自己反纾困的原则。不过可以猜到，人总是会被自己的欲望和恐惧所打败，“当上希腊总理、搬进马克西莫斯官邸”的诱惑实在难以抗拒，“被踢出欧元区”的威胁也着实有效。

2012年春天，第二轮纾困贷款在临时政府中获得批准。2012.06-2015.01新民主党为首组建的联合政府上台执政。

纾困2.0的新机制塑造了比上一版本更加“邪恶”的政府。首先是救助银行家的机制：贷款里约410-500亿欧元会通过新成立的“希腊金融稳定基金”给到希腊银行家来维持他们的生存，而这些债务将由纳税人承担，可是代表公众的议会却不能对破产银行和这项基金进行监管。其次是管理政府税收和海关的机制：这些部门的负责人将由“三驾马车”来任命和解雇，不再向本国政府和立法机构负责，反过来政府却需要为他们的不当行为负责。最后就是私有化的机制：为了债主的利益，“三驾马车”主导的一家独立机构，将尽可能多地抛售希腊的家底部门（从港口铁路到海滩小岛应有尽有），然而经济萧条下大幅贬值的公共资产只会被白白从希腊人民手中剥夺而换不了几个钱去填那无底的黑洞。这些新机制最终架空了议会对上述三大支柱的管辖权，损害了民主主权。

于是，无法破产的希腊在一轮轮虚假的纾困和愈加严厉的紧缩措施中，永远被锁进“债主监牢”，把不断减少的国民收入无力地填进不断扩大的黑洞里，最终成为制度化的债务殖民地。
